# Code Samples Directory

This directory contains all code samples from the training modules, organized by module and lesson.

## Structure

```
code-samples/
  module-0/
    lesson-1/
      sample-1.cfm
      sample-2.cfm
    lesson-2/
      sample-1.cfm
  module-1/
    lesson-1/
      sample-1.cfm
      ...
```

## Usage

1. **Run in Browser**: Click "Run" button in the training platform to execute the code
2. **Open in IDE**: Use the file path shown or open directly from your IDE
3. **Modify & Test**: Edit the files and refresh your browser to see changes

## Accessing Files

If running via CommandBox:
- URL: `http://127.0.0.1:8080/code-samples/module-X/lesson-Y/sample.cfm`

If running via ColdFusion Server:
- URL: `http://127.0.0.1:8500/code-samples/module-X/lesson-Y/sample.cfm`
