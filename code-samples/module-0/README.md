## Usage

1. Open any `.cfm` file in your IDE
2. Modify and experiment with the code
3. Run in your ColdFusion server to see output
4. Use as reference for your own projects

## Prerequisites

- ColdFusion 2021, 2023, or 2025 installed
- CommandBox (recommended) or full ColdFusion server
- IDE with ColdFusion support (VS Code, IntelliJ, etc.)
